- (constraint) Magic-link only; no passwords anywhere; login uses single-use stored tokens (AD-6).
- (constraint) Email actions: signed GET renders confirm page, state change on POST (scanner/prefetch-safe) (AD-6).
- (constraint) Fixed 9:00 AM America/New_York send (DST-tracking); no per-user send time in v1; all scheduling math in NY calendar date (AD-7).
- (constraint) Never send a broken/empty/fabricated digest — weather API down => send nothing that day; retry delivery <=3x in-run then defer to next day (AD-4).
- (constraint) Idempotent send: DB unique index (trip_id, send_date); claim-first, fetch-once-persist-then-send (AD-3); EmailLog is single per-send source of truth, forecast cached nowhere else (AD-9).
- (constraint) Geocode exactly once at creation; trip cannot exist without coords; weather fetched by coordinates only (AD-1, AD-8); account+trip created atomically, no orphan trips (AD-10).
- (constraint) External I/O only through ports (Weather/Geocoder/Mailer behind interfaces) for swappability (AD-1).
- (constraint) Per-trip send is a dispatchable Job; command only selects+dispatches (named scaling seam) (AD-2).
- (constraint) Single cadence predicate is the authority for due-ness; one Trip status owner + status-agnostic completion sweep; delete is soft delete (AD-5, AD-11).
- (constraint) Unsubscribe is account-level suppression across all a user's trips (CAN-SPAM one-click) (AD-13).
- (constraint) Deliverability: authenticated subdomain SPF+DKIM+DMARC, List-Unsubscribe one-click, stable physical postal address, spam-complaint rate <0.3%, plain-text mirror on every send.
- (constraint) Email must be fully legible with all images blocked; hosted PNG glyphs (no inline SVG/icon fonts); info never in image alone; single-column table layout <=600px, inline styles, dark-mode-aware.
- (constraint) Never alarmist: weather stated calmly (rain = soft blue, no red/amber severe styling); weather verdict never in subject line; calm concierge voice.
- (constraint) Daily run reports own liveness (heartbeat/dead-man's-switch) — a missed run alerts the builder out-of-band (AD-14).
- (constraint) Forecast shows both deg-F and deg-C with tabular numerals; conversion at render; forecast rows in destination-local calendar days.
- (constraint) WCAG 2.2 AA across email and web; status/conditions never by color alone.
